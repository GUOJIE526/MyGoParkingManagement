# 修復的方式

### 前置

目前在專案內已經建好 Area 的資料夾, 也將以前 www/root 內的所有東西都加進來了
如果有什麼東西是放在 root 外的可能要自己注意

### 可以不照我的, 但我大致流程是:

1. 直接將之前的 metadata 或 viewmodel 類引入, 將變動的地方修改
2. 把你之前 area 的東西放回來, 看看哪邊有錯誤
